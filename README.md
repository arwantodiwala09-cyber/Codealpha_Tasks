# 📖 Overview

TaskFlow is a full-stack collaborative project management platform built to simplify task planning, team collaboration, and project tracking.

It combines Kanban boards, real-time messaging, notifications, role-based authentication, analytics, file sharing, and workspace management into a single application.

The application focuses on providing a smooth user experience with modern UI/UX while supporting real-time collaboration using Socket.IO.

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/arwantodiwala09-cyber/Codealpha_Tasks.git

cd Codealpha_Tasks
```

---

## Backend Setup

```bash
cd backend

npm install
```

Create a `.env` file inside the backend directory.

```env
PORT=5000

MONGO_URI=YOUR_MONGODB_CONNECTION_STRING

JWT_SECRET=YOUR_SECRET_KEY
```

Run backend

```bash
npm start
```

---

## Frontend Setup

```bash
cd frontend

npm install

npm run dev
```
